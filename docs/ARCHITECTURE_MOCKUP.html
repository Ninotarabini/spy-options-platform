Resumen Ejecutivo

Este documento técnico describe la arquitectura de la plataforma SPY Options, diseñada para procesar datos de mercado de alta frecuencia con latencia sub-segundo. El sistema utiliza un modelo de Control Plane en la nube (Azure) para gestión y Data Plane en el Edge (K3s Local) para el procesamiento crítico.

Block 2: Arquitectura de Flujo de Datos

La plataforma prioriza la visualización instantánea mediante el flujo "Broadcast-First".

graph LR IBKR[IBKR Feed] --> Edge[Edge Processor] Edge -->|SignalR| UI[Dashboard Live] Edge -->|Async| DB[(Azure Storage)] style Edge fill:#0078d4,stroke:#fff,color:#fff style UI fill:#00f2fe,stroke:#fff,color:#000
